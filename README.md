# Gift Galore
A MERN-based full stack e-commerce platform where users can explore a variety of gifting options, add them to their cart, and securely checkout. The application supports role-based admin access, dynamic cart management, and automated email notifications.

## 🚀 Features
### User Features

Browse products by categories (cakes, bouquets, chocolates, plants, combos)

Add, update, and delete items in the cart with dynamic total calculation

Secure JWT-based user authentication & protected routes

View detailed product pages and seamless shopping experience

### Admin Features

Role-based admin panel to manage users and products

Add/update/delete products and monitor user activity

### Notifications

Automated transactional emails using Nodemailer and Node-Cron

## 🛠️ Tech Stack
Frontend: React.js, CSS

Backend: Node.js, Express.js, REST APIs

Database: MongoDB (with Mongoose)

Authentication: JWT (JSON Web Tokens)

Email Service: Nodemailer with Node-Cron

Deployment: (Netlify for frontend, Render for backend)
